# Credit Card Search
Simple python script designed to extract text from any file and find known credit card patterns (defined in regexcard.csv).
Still in development stage

**Tested file types:**
* txt
* csv
* xls   --> Using xlrd
* xlsx  --> Using xlrd
* rtf
* xml
* html
* json

**Currently unsupported file types:**
* doc
* docx
* pdf
* pptx
* zip
* jpg
* gif
* png
* mp3
* mp4
* wav
